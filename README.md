
# Mohammed Shafi Ganie Portfolio

This is a personal portfolio website built using **React** and **Tailwind CSS**. It includes:

- Hero Section
- About Me
- Skills
- Education
- Featured Project
- Contact

## Setup

```bash
npm install
npm start
```

## Deployment

Deploy easily using **Vercel**, **Netlify**, or **GitHub Pages**.
"# portfolio-site"
